The network wall plug double is one of the more important buildings.
The network wall plug double simply allows to connect more distant network connectors with each other and so to spread out the network mainly in-doors and near buildings.
Also interesting is, this pole allows to connect multiple cables at the same time.
You can also distribute now the network circuit through walls thanks to this plug.
